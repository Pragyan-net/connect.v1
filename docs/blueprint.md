# **App Name**: V1.Connect

## Core Features:

- Contact Information Display: Showcase contact links including Instagram, Discord, GitHub, and Email.
- Sticky Navigation Bar: Implement a sticky, pill-shaped navigation bar that remains at the top of the page during scrolling, offering links to 'Origin', 'Contact', 'Events', and 'Projects'.
- Full-Screen Hero Section: Create a captivating full-screen hero section with a black background and shining neon text for 'The Initiative.V1'.

## Style Guidelines:

- Background color: Light gray (#D3D3D3) for a clean and modern feel, contrasting the dark hero section.
- Card Background Color: Use #393E46 for card backgrounds.
- Page Background Color: Use #929AAB as a background under the hero section.
- Headline font: 'Space Grotesk' sans-serif for headlines, offering a techy, scientific feel. Body font: 'Inter' sans-serif for a modern look.
- Implement a full-screen hero section followed by clearly structured contact information.
- Add subtle neon-like animations to the hero text to enhance its shining effect.
